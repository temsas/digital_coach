Прототип: Помощник по цифровой грамотности

Запуск: 
Токен заменить в файле .env (Смены модели нет)
Запустить app.py






Возможные проблемы(в случае дальнейшего программирования): Долгая загрузка тестов, при теории страница не поднимается наверх(random), могут пролазить мелкие ошибки в интерфейсе оформления текста, нет архитектуры. Большой разницы между моделями не замечено, смена модели не стоит того в данном проекте на данном уровне.
